This project focuses on simplifying the control of pneumatically actuated haptic interfaces through the use of soft fluidic logic systems.

The approach replaces traditional electro-pneumatic control architectures with networks of soft bistable valves that implement logic operations directly in the fluidic domain. By arranging these valves into programmable circuits, the system can generate complex actuation patterns from a small number of input signals.

Inspired by prior work on soft fluidic logic, including bistable valve designs, this framework enables the routing and modulation of pressure signals without relying on extensive electronic control hardware. As a result, multi-degree-of-freedom haptic systems can be controlled using simpler, more scalable architectures.

This work highlights a key design direction in soft robotics: shifting complexity from software and electronics into the physical system itself, enabling more robust and deployable interfaces.
