This project explores inflatable shape-changing interfaces through the development of a flexible soft growing pin for high-extension display applications.

Inspired by shape-changing pin displays and soft growing robots, the device was designed to achieve large extension while remaining compact in its unactuated state. Each pin consists of a rigid airtight chamber, an inflatable fabric body, and a passive spring-actuated reel mechanism that enables controlled extension with passive retraction.

The resulting system provides a simple and scalable approach for creating shape-changing inflatable displays with large geometric transformation. This work connects soft robotic actuation with interface design, with potential applications in tangible displays, haptics, and adaptive physical interaction.
