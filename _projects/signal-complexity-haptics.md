This project investigates how the complexity of haptic signals influences both human perception and task performance in navigation-related scenarios.

Using an inflatable soft haptic display, we generated signals with varying spatial encoding complexity and evaluated user responses in both static perception tasks and dynamic, task-based settings. The goal was to understand how differences in signal structure affect not only how users perceive feedback, but how effectively they can use it during interaction.

Results highlight a key distinction between perceptual discrimination and practical usability. Signals that are easier to distinguish in isolation do not always lead to improved performance in real tasks, particularly when users must interpret feedback under cognitive load.

This work contributes to the design of haptic interfaces by emphasizing task-oriented evaluation and the importance of aligning signal complexity with user capabilities and application context.
