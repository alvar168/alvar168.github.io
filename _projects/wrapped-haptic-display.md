This project explores soft, wrapped haptic displays mounted directly on robot arms to provide real-time feedback during human–robot interaction.

The displays are placed at the physical interface between the user and the robot, enabling co-located feedback as the robot learns from human demonstrations. This approach contrasts with traditional interfaces by embedding feedback into the robot body itself, reducing cognitive load and improving interaction efficiency.

Through user studies, we evaluated how different feedback modalities influence teaching performance and user behavior. Results showed that co-located haptic feedback can improve usability and reduce distraction compared to more traditional interfaces.

This work contributes to the design of human-centered robotic systems by integrating feedback directly into the physical interaction loop.
