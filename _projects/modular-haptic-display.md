This project introduces a modular soft haptic display that combines reconfigurable pneumatic hardware with an information-theoretic personalization algorithm to tailor feedback signals, including pressure, frequency, and contact area, for different users and tasks.

The system was designed to support adaptable, multimodal information transfer in human–machine interaction. By combining hardware modularity with algorithmic personalization, the platform enables user-specific haptic configurations rather than relying on a single fixed interface design.

User studies showed that the modular and multimodal design supports flexible and effective haptic interaction across users. This project reflects my broader interest in soft robotic interfaces that integrate mechanical design, control, and human-centered evaluation.
